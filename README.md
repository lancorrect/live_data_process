# 直播数据处理

`1_downloadwav_by_time.py`：下载直播(flv)，并转成音频(wav)

`2_split_by_category.py`：按照类别分类

`3_asr_and_segment.py`：请求asr服务，并记录asr结果

`4_assign_segments.py`：切割音频并分配到指定文件夹

# How to run

按着顺序运行即可

```bash
python filename.py
```
